# CODICE

Repositorio principal del proyecto CODICE.

## Estructura del Proyecto

Este es un monorepo que contiene los siguientes componentes:

*   **`backend/`**: Servidor/API en NestJS.
*   **`web/`**: Aplicación web frontend en Next.js.
*   **`mobile/`**: Aplicación móvil en React Native / Expo.

## Guía de Inicio

Para empezar a trabajar con el proyecto, consulta la [Guía de Inicio Rápido](GUIA_INICIO.md).
